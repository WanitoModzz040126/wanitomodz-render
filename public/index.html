01

Setting up Shizuku

  1. Install Shizuku from Google Play or GitHub Releases.
  2. Enable Developer Options (tap Build Number 7 times).
  3. Enable Wireless Debugging in Developer Options.
  4. Open Shizuku → tap "Pairing with Wireless ADB".
  5. Start Shizuku service — status should turn green.
  6. Open WanitoModz Tool → Dashboard will show Shizuku: ACTIVE.
02

Patching Skins & Config

  1. Close MLBB completely before patching.
  2. Go to Config tab and select a Drone View preset.
  3. Go to Skin tab, select hero category and skin.
  4. Wait for Patch Success notification.
  5. Launch MLBB normally — patches take effect.
  6. Use Backup / Restore if a game update resets patches.
